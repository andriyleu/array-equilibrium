¿Que es el indicide de equilbrio?

El índice de equilibrio de un Array es cualquier entero P tal que 0 s P < N y la suma de los elementos de los índices inferiores es igual a la suma de los elementos de los índices superiores.

Por ej:

A[0] + A[1] + ... + A[P-1] = A[P+1] + ... + A[N-2] + A[N-1]

La suma de 0 elementos se asume que es igual a 0. Esto puede ocurrir si P = 0 o si P = N-1.

Ejemplos

Por ej. considerando el siguiente array A que está compuesto por 7 elementos.

A[0] = -7      A[1] = 1     A[2] = 5      A[3] = 2      A[4] = -4      A[5] = 3      A[6] = 0

P = 3 es un índice de equilibno de este array ya que:

A[0] + A[1] * A[2] = A[4] * A[5] + A[6]

P = 6 también es un índice de equilibrio de este array ya que:

A[0] + A[1] + A[2] + A[3] * A[4] * A[5] = 0
